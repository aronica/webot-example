
              <p/><p>Given a binary tree containing digits from <code>0-9</code> only, each root-to-leaf path could represent a number.</p>&#13;
<p>An example is the root-to-leaf path <code>1-&gt;2-&gt;3</code> which represents the number <code>123</code>.</p>&#13;
&#13;
<p>Find the total sum of all root-to-leaf numbers.</p>&#13;
&#13;
<p>For example,&#13;
</p><pre>&#13;
    1&#13;
   / \&#13;
  2   3&#13;
</pre>&#13;
&#13;
<p>&#13;
The root-to-leaf path <code>1-&gt;2</code> represents the number <code>12</code>.<br/>&#13;
The root-to-leaf path <code>1-&gt;3</code> represents the number <code>13</code>.&#13;
</p>&#13;
<p>&#13;
Return the sum = 12 + 13 = <code>25</code>.&#13;
</p>
