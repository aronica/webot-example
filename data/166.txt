
              <p/><p>Given two integers representing the numerator and denominator of a fraction, return the fraction in string format.</p>&#13;
&#13;
<p>If the fractional part is repeating, enclose the repeating part in parentheses.</p>&#13;
<p>&#13;
For example,&#13;
</p><ul>&#13;
<li>Given numerator = 1, denominator = 2, return "0.5".</li>&#13;
<li>Given numerator = 2, denominator = 1, return "2".</li>&#13;
<li>Given numerator = 2, denominator = 3, return "0.(6)".</li>&#13;
</ul>&#13;
&#13;
