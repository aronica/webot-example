
              <p/><p>Validate if a given string is numeric.</p>&#13;
&#13;
<p>&#13;
Some examples:<br/>&#13;
<code>"0"</code> =&gt; <code>true</code><br/>&#13;
<code>"   0.1  "</code> =&gt; <code>true</code><br/>&#13;
<code>"abc"</code> =&gt; <code>false</code><br/>&#13;
<code>"1 a"</code> =&gt; <code>false</code><br/>&#13;
<code>"2e10"</code> =&gt; <code>true</code><br/>&#13;
</p>&#13;
&#13;
<p><b>Note:</b> It is intended for the problem statement to be ambiguous. You should gather all requirements up front before implementing one.&#13;
</p>&#13;
