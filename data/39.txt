
              <p/><p>&#13;
Given a set of candidate numbers (<b><i>C</i></b>) and a target number (<b><i>T</i></b>), find all unique combinations in <b><i>C</i></b> where the candidate numbers sums to <b><i>T</i></b>. &#13;
</p>&#13;
&#13;
<p>The <b>same</b> repeated number may be chosen from <b><i>C</i></b> unlimited number of times.&#13;
</p>&#13;
&#13;
<p><b>Note:</b><br/>&#13;
</p><ul>&#13;
<li>All numbers (including target) will be positive integers.</li>&#13;
<li>Elements in a combination (<i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, … , <i>a</i><sub>k</sub>) must be in non-descending order. (ie, <i>a</i><sub>1</sub> ≤ <i>a</i><sub>2</sub> ≤ … ≤ <i>a</i><sub>k</sub>).</li>&#13;
<li>The solution set must not contain duplicate combinations.</li>&#13;
</ul>&#13;
&#13;
&#13;
<p>&#13;
For example, given candidate set <code>2,3,6,7</code> and target <code>7</code>, <br/>&#13;
A solution set is: <br/>&#13;
<code>[7]</code> <br/>&#13;
<code>[2, 2, 3]</code> <br/>&#13;
</p>&#13;
