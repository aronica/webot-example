
              <p/><p>&#13;
Given a string <i>s</i> and a dictionary of words <i>dict</i>, add spaces in <i>s</i> to construct a sentence where each word is a valid dictionary word.&#13;
</p>&#13;
&#13;
<p>&#13;
Return all such possible sentences.&#13;
</p>&#13;
&#13;
<p>&#13;
For example, given<br/>&#13;
<i>s</i> = <code>"catsanddog"</code>,<br/>&#13;
<i>dict</i> = <code>["cat", "cats", "and", "sand", "dog"]</code>.&#13;
</p>&#13;
&#13;
<p>&#13;
A solution is <code>["cats and dog", "cat sand dog"]</code>.&#13;
</p>&#13;
