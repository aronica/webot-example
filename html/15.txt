
              <p/><p>Given an array <i>S</i> of <i>n</i> integers, are there elements <i>a</i>, <i>b</i>, <i>c</i> in <i>S</i> such that <i>a</i> + <i>b</i> + <i>c</i> = 0? Find all unique triplets in the array which gives the sum of zero.</p>&#13;
&#13;
<p><b>Note:</b><br/>&#13;
</p><ul>&#13;
<li>Elements in a triplet (<i>a</i>,<i>b</i>,<i>c</i>) must be in non-descending order. (ie, <i>a</i> ≤ <i>b</i> ≤ <i>c</i>)</li>&#13;
<li>The solution set must not contain duplicate triplets.</li>&#13;
</ul>&#13;
&#13;
&#13;
<pre>&#13;
    For example, given array S = {-1 0 1 2 -1 -4},&#13;
&#13;
    A solution set is:&#13;
    (-1, 0, 1)&#13;
    (-1, -1, 2)&#13;
</pre>
