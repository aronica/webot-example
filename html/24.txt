
              <p/><p>&#13;
Given a linked list, swap every two adjacent nodes and return its head.&#13;
</p>&#13;
&#13;
<p>&#13;
For example,<br/>&#13;
Given <code>1-&gt;2-&gt;3-&gt;4</code>, you should return the list as <code>2-&gt;1-&gt;4-&gt;3</code>.&#13;
</p>&#13;
&#13;
<p>&#13;
Your algorithm should use only constant space. You may <b>not</b> modify the values in the list, only nodes itself can be changed.&#13;
</p>
