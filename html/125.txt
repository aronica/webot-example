
              <p/><p>&#13;
Given a string, determine if it is a palindrome, considering only alphanumeric characters and ignoring cases.&#13;
</p>&#13;
&#13;
<p>&#13;
For example,<br/>&#13;
<code>"A man, a plan, a canal: Panama"</code> is a palindrome.<br/>&#13;
<code>"race a car"</code> is <i>not</i> a palindrome.&#13;
</p>&#13;
&#13;
<p>&#13;
<b>Note:</b><br/>&#13;
Have you consider that the string might be empty? This is a good question to ask during an interview.</p>&#13;
<p>&#13;
For the purpose of this problem, we define empty string as valid palindrome.&#13;
</p>
