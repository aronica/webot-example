
              <p/><p>&#13;
Given a set of distinct integers, <i>S</i>, return all possible subsets.&#13;
</p>&#13;
<p><b>Note:</b><br/>&#13;
</p><ul>&#13;
<li>Elements in a subset must be in non-descending order.</li>&#13;
<li>The solution set must not contain duplicate subsets.</li>&#13;
</ul>&#13;
&#13;
<p>&#13;
For example,<br/>&#13;
If <b><i>S</i></b> = <code>[1,2,3]</code>, a solution is:&#13;
</p>&#13;
&#13;
<pre>&#13;
[&#13;
  [3],&#13;
  [1],&#13;
  [2],&#13;
  [1,2,3],&#13;
  [1,3],&#13;
  [2,3],&#13;
  [1,2],&#13;
  []&#13;
]&#13;
</pre>
