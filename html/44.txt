
              <p/><p>Implement wildcard pattern matching with support for <code>'?'</code> and <code>'*'</code>.</p>&#13;
&#13;
<pre>&#13;
'?' Matches any single character.&#13;
'*' Matches any sequence of characters (including the empty sequence).&#13;
&#13;
The matching should cover the <b>entire</b> input string (not partial).&#13;
&#13;
The function prototype should be:&#13;
bool isMatch(const char *s, const char *p)&#13;
&#13;
Some examples:&#13;
isMatch("aa","a") → false&#13;
isMatch("aa","aa") → true&#13;
isMatch("aaa","aa") → false&#13;
isMatch("aa", "*") → true&#13;
isMatch("aa", "a*") → true&#13;
isMatch("ab", "?*") → true&#13;
isMatch("aab", "c*a*b") → false&#13;
</pre>
